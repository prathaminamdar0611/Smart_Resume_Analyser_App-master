# Smart_Resume_Analyser_App-master
Smart Resume Analyser is a web-based application that allows users to upload resumes and automatically extract important information such as skills, education, experience, and keywords. The system processes documents asynchronously, provides real-time progress updates, and allows users to review, edit, and export the analyzed data.
